# Thank you, CERAGEM (v2 리뉴얼)

세라제머십 7가지 가치 기반 사내 동료 칭찬/포인트 플랫폼.

## 실행 방법
```bash
npm install
npm run dev      # 개발 서버
npm run build    # 프로덕션 빌드 (Vercel 배포용)
```

## 데모 계정
| 역할 | 아이디(이름/사번) | 비밀번호 |
|---|---|---|
| 관리자 | 이동현 / 100001 | 1234 |
| 일반 | 김소영 / 100002 | 0000 (최초 로그인 시 변경) |

## 구조
- `src/types.ts` — 도메인 타입, 세라제머십 7 정의, 포인트 정책 상수
- `src/store/store.ts` — localStorage 기반 스토어. 포인트는 잔액 저장이 아닌 **원장(Ledger) 합산 방식**으로 계산
- `src/pages/` — 로그인/홈/보내기/피드/히스토리/랭킹/가이드/관리자
- `src/styles.css` — 디자인 토큰 (코랄레드 + 골드 + 뉴트럴, PRD §6)

## 포인트 정책 (PRD §4)
- 매월 5,000P 보내기 포인트, 미사용 소멸 (월 기준 자동 계산)
- 동일 수신자 월 최대 1,000P
- 받은 포인트 = 리워드(현금 교환용) + 아너(분기 시상용) 동시 적립
- 현금 교환해도 분기 누적은 유지

## 배포
Vercel에 그대로 import하면 됩니다 (Framework: Vite).
현재 데이터는 브라우저 localStorage에 저장되는 프로토타입입니다.
실서비스 전 백엔드/DB 연동 및 인사·재무 검토(현금 교환 과세)가 필요합니다.
